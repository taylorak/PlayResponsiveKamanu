PlayResponsiveKamanu

A mockup of the Kamanu Composites website using Twitter Bootstrap, Play, and responsive design principles.

Download this project as a .zip file Download this project as a tar.gz file

Overview

This is a Play application that implements a mockup of the Kamanu Composites Website using Twitter Bootstrap 3 and responsive design principles.

Installation

git clone https://github.com/taylorak/ResponsiveKamanu.git  
cd ResponsiveKamanu  
play  
run

View the running application at http://localhost:9000

Usage

No functionality because this is a UI mockup
